# DSP_task-1

## Overview

This project contains digital signal processing implementations for ECG, EEG, Doppler, and SAR Images.

## Features

- Signal visualization
- Data analysis tools with AI models

## Getting Started

### Prerequisites

- Python 3.x
- Required dependencies (see `requirements.txt`)

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```
